Парсинг вакансий по url работает только с шаблоном https://rabota.yandex.ru/...
При желании можно реализовать дополнительные Service для других сайтов с вакансиями.

Фильтр работает по заголовку. Необходимо ввести вакансию в поле и нажать на кнопку.
Для сброса фильтра произвести поиск по пустому заголовку.

При многократном обращении Яндекс блокирует, выдает html для парсинга без вакансий.
Очень неудобно тестировать программу при написаниии кода.
Чтобы избавится от этого, есть закоменченный код в классе ServiceYaRab.
Данный код загружает Document из файла на диске.
Если его разкоментить, а код создания Document из url - закоментить, то можно избавится от блокировки Яндекса при тестировании.

        Например: контакты, телефоны, email;
 - реализовать дополнительные Service для других сайтов с вакансиями;
